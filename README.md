# Pandoc 模板

自用的一些 Pandoc 模板

## 使用方法

在 Windows 上，克隆本仓库到 `%APPDATA%\pandoc` 路径下：

    git clone https://ghproxy.com/https://github.com/duzyn/pandoc-templates \
    "%APPDATA%\pandoc"

在 Linux 或 macOS 上，克隆本仓库到 `~/.local/share/pandoc` 路径下：

    git clone https://ghproxy.com/https://github.com/duzyn/pandoc-templates \
    ~/.local/share/pandoc
